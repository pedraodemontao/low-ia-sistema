# Template de Oferta

Copie esta pasta para `ofertas/{nome-da-oferta}/` e preencha etapa por etapa.

Cada arquivo = uma etapa do pipeline (ver `/pipeline.md`). Preencha em ordem; cada etapa usa a anterior.

| Arquivo | Etapa |
|---------|-------|
| `00-alvo.md` | Alvo / nicho |
| `01-espionagem.md` | Achar oferta validada (FB Ad Library) |
| `02-coleta.md` | Scrape Apify |
| `03-analise.md` | Análise do scrape |
| `04-modelagem.md` | Estrutura vencedora |
| `05-diferenciacao.md` | ★ Reestruturação (foge do leilão) |
| `06-imagens/` | Criativos (Higgsfield) |
| `07-paginas/` | LP / checkout / upsell |
| `08-copy.md` | Copy agressivo/urgente |
| `09-disparo.md` | WaDisparo + tráfego |
